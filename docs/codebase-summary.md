## File Details

### VoxType/App/VoxTypeApp.swift (69 lines)

Entry point. Two types:

- **`VoxTypeApp: App`** -- `@main` struct. Creates `AppController` as `@StateObject`. Shows empty `Settings` scene.
- **`AppController: ObservableObject`** -- `@MainActor`. Owns all service instances. Wires Combine bindings from `DictationManager` to `MenuBarController` and `HUDController`. Calls `TranscriptionService.prepareModel()` in background `Task` on init.

### VoxType/App/MenuBarController.swift (32 lines)

- **`MenuBarController`** -- Creates `NSStatusItem` with "mic" SF Symbol. Contains "Quit VoxType" menu item. `updateIcon(for:)` switches SF Symbol based on `DictationState`: mic (idle), mic.fill (listening), mic.badge.xmark (transcribing), checkmark.circle (done), exclamationmark.triangle (error).

### VoxType/Features/DictationState.swift (14 lines)

- **`DictationState: Equatable`** -- Enum with associated values: `.idle`, `.listening`, `.transcribing`, `.done(text: String)`, `.error(message: String)`. Computed `isProcessing` returns true for listening/transcribing.

### VoxType/Features/DictationManager.swift (130 lines)

- **`DictationManager: ObservableObject`** -- `@MainActor`. Orchestrates full dictation lifecycle.
  - Published: `state`, `audioLevel`
  - `bindHotkey()`: subscribes to HotkeyManager keyPressed/keyReleased
  - `startDictation()`: precondition guards, calls `audioService.startRecording()`
  - `stopDictation()`: calls `audioService.stopRecording()`, spawns async transcription
  - `transcribeAndInsert(_:)`: chains TranscriptionService -> TextInsertionService, logs latency via ContinuousClock
  - `scheduleReset()`: 3-second Task.sleep then reset to idle

### VoxType/Services/AudioCaptureService.swift (158 lines)

- **`AudioCaptureService: ObservableObject`** -- Not @MainActor. Manages AVAudioEngine.
  - Target format: 16kHz mono Float32
  - `startRecording()`: creates engine, installs tap with `TapHandler` inner class for format conversion
  - `TapHandler`: @unchecked Sendable, calculates RMS level, converts via AVAudioConverter, appends to pcmBuffers
  - `stopRecording()`: removes tap, stops engine, calls `mergeBuffers()`
  - `mergeBuffers()`: concatenates all PCM buffers into single buffer
- **`AudioError`** -- enum: formatError, notRecording

### VoxType/Services/TranscriptionService.swift (115 lines)

- **`TranscriptionService: ObservableObject`** -- `@MainActor`. Manages WhisperKit.
  - Model: `large-v3-turbo`
  - `prepareModel()`: downloads and loads WhisperKit model
  - `transcribe(_:)`: writes PCM to temp WAV, calls `whisperKit.transcribe(audioPath:)`, deletes temp file
  - `writePCMTempoWAV()`: manual RIFF header + IEEE Float32 data
  - Published `modelStatus: ModelStatus` (notLoaded, downloading, loading, ready, failed)
- **`TranscriptionError`** -- enum: modelNotReady, invalidAudio, emptyResult

### VoxType/Services/HotkeyManager.swift (126 lines)

- **`HotkeyManager: ObservableObject`** -- Global hotkey via CGEvent tap.
  - Target: keyCode 61 (Right Option)
  - `keyPressed` / `keyReleased`: PassthroughSubject<Void, Never>
  - `installEventTap()`: CGEvent.tapCreate on cgSessionEventTap, listenOnly
  - `handleEvent()`: filters for target keyCode, tracks isKeyDown state, re-enables tap on timeout
  - Requires Accessibility permission (AXIsProcessTrusted)

### VoxType/Services/TextInsertionService.swift (81 lines)

- **`TextInsertionService`** -- Clipboard paste and Cmd+V simulation.
  - `insertText(_:)`: saves clipboard, writes text, 50ms delay, simulates Cmd+V, schedules restore after 500ms
  - `simulatePaste()`: CGEvent keyDown/keyUp for virtual key 0x09 with .maskCommand
  - Returns `InsertionResult`: success, clipboardOnly, failed(String)

### VoxType/UI/HUDView.swift (178 lines)

Four types in one file:

- **`HUDPanel: NSPanel`** -- Borderless, non-activating, floating level, clear background, all spaces. 260x52pt frame.
- **`HUDController`** -- `@MainActor`. Manages panel lifecycle. `show(state:audioLevel:)` creates panel lazily, positions near menu bar center. `updateAudioLevel(_:)` re-renders during listening.
- **`HUDContentView: View`** -- SwiftUI. Icon + label + optional audio bar / result preview / error message. Uses `.ultraThinMaterial` background with rounded corners.
- **`AudioLevelBar: View`** -- Red progress bar proportional to RMS level (min 2pt width, 4pt height).

### project.yml (75 lines)

XcodeGen configuration. Key settings:

- `SWIFT_VERSION: "5.0"`, `SWIFT_STRICT_CONCURRENCY: minimal`

## Dependency Graph

```
VoxTypeApp
  └── AppController
        ├── MenuBarController
        ├── HUDController
        │     └── HUDPanel + HUDContentView + AudioLevelBar
        └── DictationManager
              ├── AudioCaptureService
              ├── TranscriptionService  (-> WhisperKit)
              ├── TextInsertionService
              └── HotkeyManager
```

All services are instantiated by `AppController` and injected into `DictationManager` via init parameters. No service discovers another service -- all wiring is explicit.

## Key Protocols / Patterns

| Pattern | Usage |
|---|---|
| ObservableObject + @Published | State propagation (DictationManager, TranscriptionService, AudioCaptureService) |
| PassthroughSubject | Event signaling (HotkeyManager key events) |
| Combine sink | Binding services to UI (AppController.wireUI) |
| @MainActor | UI-bound classes (AppController, DictationManager, TranscriptionService, HUDController) |
| DispatchQueue | Audio buffer thread safety (AudioCaptureService.bufferQueue) |
| async/await | WhisperKit transcription, text insertion delays |
| Delegation via init injection | DictationManager receives all services as constructor parameters |
